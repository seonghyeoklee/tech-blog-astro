힙(Heap)은 런타임에 동적으로 할당되는 메모리 영역입니다. new 키워드로 생성한 객체가 이곳에 저장됩니다.

## 저장되는 것

- new로 생성한 객체
- 배열
- 인스턴스 변수

## 특징

| 항목 | 설명 |
|------|------|
| 할당 속도 | 상대적으로 느림 |
| 해제 | GC 또는 명시적 해제 |
| 크기 | 큼 (GB 단위 가능) |
| 스레드 | 모든 스레드가 공유 |

## OutOfMemoryError

힙 공간이 부족하면 발생합니다. 객체를 너무 많이 생성하거나 메모리 누수가 원인입니다.

## Java 힙 구조

```
┌─────────────────────┐
│  Young Generation   │ ← 새 객체, Minor GC
├─────────────────────┤
│  Old Generation     │ ← 오래된 객체, Major GC
└─────────────────────┘
```
