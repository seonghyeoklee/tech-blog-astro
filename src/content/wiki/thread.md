스레드(Thread)는 프로세스 내에서 실행되는 흐름의 단위입니다. 하나의 프로세스는 최소 하나의 스레드(메인 스레드)를 가집니다.

## 메모리 공유

같은 프로세스 내의 스레드들은 다음을 공유합니다.

- Code 영역
- Data 영역
- Heap 영역

각 스레드가 독립적으로 가지는 것은 **Stack**뿐입니다.

## 프로세스와 비교

| 구분 | 프로세스 | 스레드 |
|------|----------|--------|
| 메모리 | 독립적 | 공유 |
| 생성 비용 | 높음 | 낮음 |
| 컨텍스트 스위칭 | 느림 | 빠름 |

## 주의점

메모리를 공유하기 때문에 동기화 문제(Race Condition)가 발생할 수 있습니다. 공유 자원 접근 시 적절한 동기화 처리가 필요합니다.
