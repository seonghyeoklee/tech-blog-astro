프로세스(Process)는 디스크에 저장된 프로그램을 메모리에 올려서 실행한 상태입니다. 운영체제는 각 프로세스에게 독립된 메모리 공간을 할당합니다.

## 메모리 구조

프로세스의 메모리는 4가지 영역으로 구성됩니다.

| 영역 | 설명 |
|------|------|
| Code | 실행할 명령어 (읽기 전용) |
| Data | 전역 변수, static 변수 |
| Heap | 동적 할당 메모리 |
| Stack | 지역 변수, 함수 호출 정보 |

## 특징

- 각 프로세스는 독립된 메모리 공간을 가짐
- 다른 프로세스의 메모리에 직접 접근 불가
- 프로세스 간 통신은 IPC(Inter-Process Communication) 필요

## 관련 개념

- 스레드: 프로세스 내에서 실행되는 흐름의 단위
- 컨텍스트 스위칭: 프로세스 간 전환 시 상태 저장/복원
