PCB(Process Control Block)는 운영체제가 각 프로세스에 대한 정보를 저장하는 자료구조입니다.

## 포함 정보

| 항목 | 설명 |
|------|------|
| 프로세스 ID | 고유 식별자 |
| 프로세스 상태 | 실행, 준비, 대기 등 |
| Program Counter | 다음 실행할 명령어 주소 |
| 레지스터 값 | CPU 레지스터 상태 |
| 메모리 정보 | 페이지 테이블, 세그먼트 정보 |
| 스케줄링 정보 | 우선순위, 스케줄링 큐 포인터 |
| I/O 정보 | 열린 파일, 할당된 장치 |

## 역할

컨텍스트 스위칭 시 현재 프로세스의 상태를 PCB에 저장하고, 다음 프로세스의 PCB에서 상태를 복원합니다.
