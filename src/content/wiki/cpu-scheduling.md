CPU 스케줄링은 운영체제가 CPU 시간을 여러 프로세스에 할당하는 방법입니다.

## 주요 알고리즘

- **FCFS**: 먼저 온 순서대로 처리
- **SJF**: 실행 시간이 짧은 것부터 처리
- **Round Robin**: 시간을 나눠서 공평하게 분배
- **우선순위**: 중요한 프로세스 먼저 처리
